sudo 环境变量继承

2014-04-01

修改sudo的配置文件：

sudo visudo

将Defaults env_reset改成Defaults !env_reset,取消掉对PATH变量的重置。

然后在.bashrc中最后添加

alias sudo='sudo env PATH=$PATH'

这样sudo执行命令时所搜寻的路径就是系统的PATH变量中的路径，如想添加其他变量也是类似。
